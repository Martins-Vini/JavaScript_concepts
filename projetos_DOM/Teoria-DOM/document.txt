Aula 1 - Conceito, árvore DOM e acesso aos elementos da árvore

*DOM = Interpretador JS para o lado do navegador, criando uma árvore DOM e BOM da página e sistema aberto, sendo possível acessar esses elementos e manipular eles de alguma forma para produzir uma interação ou funcionalidade, guardando os elementos (Tags) em variáveis

- Métodos de seleção do documento
getElementById()
getElementByName()
getElementsByTagName()
querySelector()

Métodos que fazem você transitar pela árvore DOM (Ensinados na ETEC):

No DOM, elementos dentro do HTML são chamados de elementos em nó, pois tudo e qualquer coisa está ligada entre si indiretamente atraés da árvore

elementoSelecionado.children -> Pega o elemento filho ou os elementos, fazendo um array daqueles elementos

elementoSelecionado.childNode -> Retorna até elementos de texto e comentários

elementoSelecionado2.parentElement -> Pega o Elemento pai do elemento selecionado se ele estiver dentro do HTML

(Caso tente pegar elementos que excedem o HTML, é retornado nulo pra variável que esse elemento será guardada)

elementoSelecionado2.parentNode ->  Pega até elementos que excedem o HTML, retornando algo

- Métodos de seleção que pega muitos elementos de uma vez:
getElementsByClass()
queryselectorAll()

Ambos fazem uma lista com os elementos da mesma classe e manipula aquelas tags da maneira que for necessária, através de loops e métodos de array como o find(), map(), entre outros
