Operadores de comparação em js

Quais são os operadores?

  • == Igual
  • != Diferente
  • > Maior
  • < Menor
  • >= Maior ou igual
  • <= Menor ou igual

Comparações estritas

Comparações estritas são para mudar o fluxo de um programa ou comparar valores de variáveis se eles forem o mesmo e forem o mesmo tipo de dado Mais seguro e melhor opção, já que o JavaScript não é uma linguagem fortemente tipada. Além de que, sempre resultam em um boolean

  • === Estritamente igual
  • !== Estritamente diferente

Tipos falsy e truthy

Esses são tipos que vem de comparação, que representam o estado de algo que existe e coisas que existem numa comparação

Tudo que é falsy

  • -0
  • " "
  • null
  • undefined
  • NaN
