Funções e tipo de funções em JS

Uma função serve para fazer jus ao paradigma funcional de programação, basicamente separando o programa em funções, em blocos em que cada parte faz alguma coisa

Abra o arquivo JS e o console para ver mais anotações sobre o conteúdo

Sintaxe

function soma(a,b){
return a+b;
};
soma(4,5)
O retorno deve ser 9

Valor x Referência

Basicamente se você está programando, é necessário ter a ciência de que, uma variável passada como parâmetro numa função, só tem seu valor alterado dentro da função. Não tem como "vazar" o processamento daquela variável para fora.

No caso de um atributo JSON numa função, aí o caso é diferente, se um atributo de um objeto pessoa, por exemplo, "cep", for mudado dentro da função, será mudado fora da função. Podendo impactar diretamente no objetivo do programa.

Escopo da função

Uma função não funciona corretamente se ela for atribuída dentro de uma variável ou constante e for chamada acima da sua estrutura:

sum(4,5)
const sum = function (a,b){
return a+b;
};

O retorno deve ser 9
