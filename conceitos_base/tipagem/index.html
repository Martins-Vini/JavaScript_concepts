Tipos de dados e tipagem

O JavaScript é uma linguagem de baixa tipagem, ou seja, os dados só têm tipos atribuídos quando o programa está rodando, não tendo necessariamente uma pré-tipagem.

Tipos de dados em JavaScript

  • String
  • Number (Int, float, Infinity)
  • NaN
  • Boolean
  • Null
  • Undefined
  • Object
  • Array
